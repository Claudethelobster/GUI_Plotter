# Installing
> [!NOTE]
> Simplest way to run the software is to download the repo and run `src/badger_plot/__main__.py`.
For more freedom of use, best to install the package.

Can be installed into a and python environment using:
```powershell
pip install "git+https://github.com/Claudethelobster/GUI_Plotter"
```

Which can the be ran in powershell using:
```powershell
badger_plot
```
or in a python:
```python
import badger_plot

badger_plot.run()
### or
# badget_plot.main()
### or
# badger_plot.badger_plot()
```
